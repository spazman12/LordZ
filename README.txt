================================================================================
  LORDZ - Deadside Workshop Mirror Tool
================================================================================

GETTING STARTED (3 STEPS)
-------------------------

  1. Double-click:  LordZ - Start Here.bat

  2. In the app:
       - Click "Download & Install SteamCMD" (one time only)
       - Enter your Steam username
       - App ID 895400 is pre-filled for Deadside - click Verify
       - Add source mod IDs to the Mirror Queue

  3. Click "Generate Script", then "Run Script"
       - A PowerShell window opens
       - Enter your Steam password when asked
       - Approve Steam Guard on your phone if prompted


OPTIONAL
--------

  Create Desktop Shortcut.bat  - puts a LordZ icon on your desktop

  Start-LordZ-Debug.bat      - use only if the app will not start
                               (shows error messages in a console)


REQUIREMENTS
------------

  - Windows 10 or 11
  - PowerShell 5.1 (built into Windows)
  - A Steam account that owns Deadside
  - Internet connection


WHERE FILES GO
--------------

  steamcmd\          SteamCMD installs here automatically
  Generated\         Mirror scripts are saved here
  lordz.settings.json   Your settings (created on first save)
